# Statistical-Data-Analysis-And-Visualization
Overview

This project focuses on statistical analysis and data visualization using Excel and Google Sheets on 3 datasets containing 1000+ records. The project applies statistical techniques and probability concepts to identify patterns, relationships, and insights from structured datasets.

Tools Used

* Excel
* Google Sheets

Statistical Techniques

* Descriptive Statistics
* Contingency Tables
* Variance
* Standard Deviation
* Covariance
* Correlation Coefficient
* Probability Analysis
* Binomial Distribution
* Experimental Distribution

Data Visualization

* Scatter Plots
* Column Charts
* Distribution Graphs
* Comparative Analysis Charts

Variable Types Analyzed

* Nominal Variables
* Ordinal Variables
* Discrete Variables
* Continuous Variables
* Ratio Variables

Objectives

* Analyze large datasets using statistical methods
* Identify relationships between variables
* Generate analytical insights from structured data
* Visualize trends and patterns for better interpretation

Key Learnings

* Statistical analysis and interpretation
* Data visualization techniques
* Probability and distribution analysis
* Spreadsheet-based analytical reporting
* Data-driven decision making

Outcome

Successfully performed statistical analysis and visualization on multiple datasets and generated insights using descriptive statistics, probability analysis, and graphical representations.
